# DALI Docker Installer Directory

Please place the proprietary SICStus Prolog Linux x86-64 installer provided by the University in this directory before building the Docker image.

Expected filename: `sp-4.6.0-x86_64-linux-glibc2.17.tar` (or similar, ensure it matches the `SICSTUS_TARBALL` argument in the `Dockerfile`).

> **Note:** Do NOT commit the tarball to the repository. Conform to licensing terms.
