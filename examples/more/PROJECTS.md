# Additional Projects

This folder collects extra DALI Multi-Agent System examples and related projects contributed over time, many of them originating from DISIM-UnivAQ student work.

## What You Can Find Here

- derived MAS examples
- experimental student projects
- domain-specific DALI applications

These materials can be useful as inspiration, as reference implementations, or as starting points for new custom MAS setups.

Some additional examples may also be referenced through the repository submodules configuration in [.gitmodules](../../.gitmodules). Those entries point to external projects that can be cloned and explored separately.
